# SQL Practice (Beginner)

This repository contains beginner-level SQL practice questions with detailed problem statements and working solutions.

## What’s Included
- SQL queries covering joins, filtering, grouping, and subqueries
- Questions include example input and expected output
- Solutions written and tested in a Docker-based SQL environment 

## Topics Covered
- SELECT, WHERE
- LEFT JOIN
- GROUP BY and HAVING
- Subqueries and NOT IN
- Handling NULL values

## How These Were Practiced
All queries were written and executed during classroom practice using Docker.

## File Structure
- `sql_practice_questions.sql` – Contains all questions and solutions

## How to Run
QUeries can be executed in any SQL environmnet after creating the required tables.
